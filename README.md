Плагин для Firefox/Chrome браузера для слежения за динамикой цен на сайтах www.vseinstrumenti.ru,  www.ozon.ru, www.wildberries.ru,  www.avito.ru (Всеинструменты, Озон, (Вальберис, Авито с версии 1.2) ). 

# Замечания разработчика

После полу года наблюдением за цен автор пришел к выводу,что платформа озон дает 100% скидки определенным кругам лиц на пару минут. Время выбирается очень не приемленное для большинства глаз. Аддон может мпомочь вам поймать настоящую скидку. но для этого может понадобится время. (подтверждение моим слован см на фото при скачивания аддона браузера firefox)

# Инструкция пользования 

1) Зайти на страницу желаемой вещи
2) Правой клавишей мыши кликнуть по цене
3) В меню выбираете "отслеживать цену" и товар помещается в список. Список можно увидел в панели аддонов. Обновление цены происходит каждые 10 минут и в случае изменения цен вам будет выдано уведомление.
4) Скачайте готовый аддон из шапки данного руководства
5) Если вы хотите добавить tg бота,то вам нужно через botfather создать его в свое аккаунте,получить api ключ и вставить в аддон. Если при нажатии на кнопку ничего не происходит,то(!!!) зайдите в свой бот,отправьте в него /start и дальше в аддоне ещё раз введите ключ и сохраните. (решение проьлемы протестировано несколько раз)

# Внимание

WARNING: Яндекс.маркет не поддерживается (но работает, цена добавляется и отслеживается),так как они используют "антибот"

Для сборки:

```
web-ext sign --api-key="user:..." --api-secret="..." --source-dir . --channel="unlisted" --amo-metadata metadata.json -v
```

Для сборки нужно сохранить структуру и запаровать все файлы и папки ZIP(устаревшая): 

```
zip -r ../seetoprice.zip *
или
jpm xpi
```

для работы Телеграм вам нужно:
1) Напишите @BotFather в Telegram
2) Используйте команду /newbot
3) Получите токен вида 123456789:ABCdefGHIJKlmNoPQRsTUVwxyZ
4) В своем боте напишите /start (важный момент)
5) Вставьте токен в аддон и нажмите сохранить. Если при нажатии на кнопку ничего не происходит,то(!!!) зайдите в свой бот,отправьте в него /start и дальше в аддоне ещё раз введите ключ и сохраните. (решение проьлемы протестировано несколько раз)


Для теста можете попробовать режим разработчика:
```

```
